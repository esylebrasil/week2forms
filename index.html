What is Responsive Web Design?

Responsive web design is a way of designing and developing web content with one thing in mind: you never know exactly how a user will access the site. Will they access from a desktop or mobile device? Will the user have the content magnified or reduced? Will they be viewing the content in landscape or portrait mode?

There is no way to completely predict user behavior, which makes having a static design that will look good 100% of the time completely impossible.

Responsive web design, however, allows the site to be flexible to how users access the site based on the device, screen orientation, browser window size, view size, platform and many other factors by using flexible grids and layouts, images, HTML and CSS. Thus, solving the problem of “how will users view this content?” Instead of using static content that is inflexible or clunky, responsive design fluidly and seamlessly adjust to the size of the canvas at hand, bringing a delightful user experience to life on the web.
